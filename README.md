# What's Cooking, Good Looking? 🍳

## Prepared by Joshua Couto

*What's Cooking, Good Looking?* is a shared household management web application designed to assist my partner Kayla and I run our home. The application is designed to tackle one of the more exhausting tasks of running a shared household — meal planning for the week and grocery shopping.

To learn more about this application, check out the project [wiki](https://github.com/GymLeaderC/whats-cooking-good-looking/wiki/)!
